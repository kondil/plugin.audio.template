# This is the README file of the template audio plugin for kodi.

For kodi v16 and earlier we need the following contents ::

addon.py
addon.xml
fanart.jpg
icon.png
LICENSE.txt
changelog.txt
resources/
  settings.xml
  language/
  lib/
  data/
  media/

For kodi v17 and later we need the following contents ::

addon.py
addon.xml
LICENSE.txt
resources/
  settings.xml
  language/
  lib/
  data/
  media/
  fanart.jpg
  icon.png
